💖 Love Proposal Website

A fun, interactive, and emotional mini web app designed to ask a simple question:

"Do you love me?" 😁

---

🌟 Features

- 💘 Multi-step interactive flow
- 😂 “No” button runs away (fun element)
- 🎬 Smooth animations & transitions
- 🎉 Celebration on “Yes” click (confetti + hearts)
- 📱 Fully responsive (mobile friendly)
- ✨ Clean and modern UI

---

🚀 Live Experience

A playful and emotional journey where the user tries to click “No”… but the website makes it nearly impossible 😈

---

🛠️ Tech Stack

- HTML
- CSS
- JavaScript

(No frameworks used)

---

📂 How to Use

1. Download or clone the project
2. Open "index.html" in your browser
3. Enjoy the experience 😄
